# HTML5-CSS3 Practice 🌐

This repository contains practical examples and demos using **HTML5** and **CSS3** for building modern, responsive web pages.

---

## 📌 About

These files demonstrate fundamental web development concepts, including:

- Semantic HTML structure  
- Text, images, and lists  
- CSS styling and selectors  
- Layouts using Flexbox & positioning  
- Navigation menus  
- Forms and input styling  
- Responsive design basics

Each example focuses on a specific topic to help beginners learn frontend development step by step.
